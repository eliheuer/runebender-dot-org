<DocSection title="Current status">

The project README describes the editor as very alpha and actively under development. Public docs should preserve that bluntness and avoid over-specifying unstable behavior.

</DocSection>

<DocSection title="Contribution paths">

- File issues and pull requests in [eliheuer/runebender-xilem](https://github.com/eliheuer/runebender-xilem).
- Use the [Support](/docs/support.html) page to choose the right channel for issues, pull requests, and discussion.
- Keep format and architecture notes close to code until behavior stabilizes.

</DocSection>
