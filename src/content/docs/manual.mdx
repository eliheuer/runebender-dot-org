<DocSection title="Drawing" id="drawing">

Runebender Xilem exposes a focused set of editing tools for glyph construction.

- **Select tool:** select points, contours, and editable objects.
- **Pen tool:** draw cubic bezier contours.
- **Hyperbezier pen:** draw smooth paths defined by on-curve points.
- **Knife tool:** cut contours during outline editing.

</DocSection>

<DocSection title="Editing">

Current README-level operations include deletion, nudging, point type conversion, reverse direction, copy/paste, duplicate, flip, and rotate.

<Callout>Because the app is alpha, this page avoids promising exact selection states or long-term tool behavior. Use the shortcut reference for the current command map.</Callout>

</DocSection>

<DocSection title="Preview" id="preview">
