<DocSection title="Workflow">

1. Import a reference image with `Cmd/Ctrl + I`.
2. Position and scale it against the glyph metrics.
3. Press `Cmd/Ctrl + T` for local img2bez tracing.
4. Use `Cmd/Ctrl + Shift + T` to refit existing outlines.
5. Undo if needed; the background image is preserved for comparison.

</DocSection>

<DocSection title="Backends">

<table>
  <caption>Tracing backends</caption>
  <tbody>
    <tr><th>img2bez</th><td>Local bitmap-to-bezier tracing. Works offline and is the recommended default path.</td></tr>
    <tr><th>QuiverAI</th><td>Optional cloud vectorization via <code>Cmd/Ctrl + Shift + Y</code>. Requires network access, an API key, and API credits.</td></tr>
