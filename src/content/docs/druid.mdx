<DocSection title="Role">

The Druid implementation is the original public Runebender line in the Linebender organization. Xilem is a port from that earlier Rust/Druid editor, and the newer project family now includes shared Core code and the Comfy front-end path.

</DocSection>

<DocSection title="Use this when">

- You are researching project history.
- You need to understand where the Xilem port came from.
- You are comparing old Druid behavior with current front-end experiments.
