<DocSection title="Trace">

<table>
  <caption>Image workflows</caption>
  <tbody>
    <tr><th>Import image</th><td>Background image import is exposed through <code>Cmd/Ctrl + I</code>.</td></tr>
    <tr><th>Local trace</th><td>img2bez tracing is exposed through <code>Cmd/Ctrl + T</code>.</td></tr>
    <tr><th>Refit</th><td>Refit existing outlines to the background image with <code>Cmd/Ctrl + Shift + T</code>.</td></tr>
    <tr><th>Cloud trace</th><td>Optional QuiverAI tracing is exposed through <code>Cmd/Ctrl + Shift + Y</code>.</td></tr>
    <tr><th>Lock image</th><td>Background image lock toggle is listed as <code>Cmd/Ctrl + L</code>.</td></tr>
  </tbody>
</table>

</DocSection>
