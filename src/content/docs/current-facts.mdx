<DocSection title="Durable map">

- Runebender Xilem is a Rust/Xilem font editor.
- It is a port of the earlier Druid-based Runebender.
- The current workflow is source-first: clone the repo and run with Cargo.
- UFO is the current source-font workflow named by the project.
- Background image tracing exists through img2bez and optional QuiverAI.
- Hyperbezier contours are marked in GLIF with `identifier="hyperbezier"`.

</DocSection>
