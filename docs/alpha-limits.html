Known limits

Alpha means moving.

Runebender Xilem is useful as an experiment and development surface, but it should not be presented as stable production software. These limits are part of the public docs.

StatusPublic caveats
AudienceProspective users, contributors, agents
SourceProject status and current docs
StabilityExpected to change as the app matures

Interface

  • Tool behavior and keyboard shortcuts may change.
  • Docs describe the shape of workflows rather than every UI state.
  • Screenshots show current development state, not a final product surface.

Files

  • Use version control or backups when testing real UFO sources.
  • Format behavior should be verified against the current source repository.
  • Hyperbezier behavior is documented by the canonical source-repo specification.

Tracing

  • img2bez is the local tracing path.
  • QuiverAI is optional and requires network access, an API key, and API credits.
  • Traced outlines may need cleanup and metric adjustment.

Docs

These docs are intentionally broad. They should help people and agents find the right source files, commands, and concepts without freezing details that are still in flux.
