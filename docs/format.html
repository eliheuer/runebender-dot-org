UFO sources

Format notes.

Runebender is organized around source fonts that can be inspected, versioned, and moved between tools. The public docs should be explicit about what gets written.

UFO

UFO is the current source workflow named in the Runebender Xilem README. The site should document supported UFO versions, file IO expectations, save behavior, and compatibility with other tools.

Hyperbezier

Runebender Xilem supports on-curve hyperbezier paths, with control points computed by a spline solver. The source repository includes a hyperbezier UFO extension specification.

Before publishing, link this page to the canonical spec in the repo and summarize fallback behavior for tools that do not understand the extension.
