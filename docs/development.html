Development

Build the tool.

Runebender Xilem is a port of the earlier Druid-based Runebender to Xilem, a Rust UI framework from the Linebender ecosystem.

Current status

The project README describes the editor as very alpha and actively under development. Public docs should preserve that bluntness.

Contribution paths

  • File issues and pull requests in eliheuer/runebender-xilem.
  • Use the Linebender Zulip channel for development discussion.
  • Keep format and architecture notes close to code until behavior stabilizes.
